Get Started

Add scripts to your html 

                <script type="module" src="//iota-button.org/build/iota-button.esm.js"></script> 

                <script nomodule src="//iota-button.org/build/iota-button.js"></script>

Add following HTML 

              // get paid
              <iota-button address="iota1..." amount="500000000"></iota-button> 

              <iota-button address="iota1..." amount="50" currency="USD"></iota-button>

              // show balance
              <iota-button address="iota1..." type="balance"></iota-button>

              <iota-button address="iota1..." currency="EUR" label="Donated"
              type="balance"></iota-button>

About

IOTA Button can be used to accept payments, reviews, donations or just show balance in your wallet. Simple button can be embeded on any website with minimal coding. It's completely open source, client based JavaScript code so you're in control. This is a firt prototype to see if there is an interest.

Please get involved on Github if you have project where you want to use it or have ideas on how to expand this project.
